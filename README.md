# 🥐 Piekarzynka (Client)

Aplikacja mobilna/webowa dla klientów piekarni online, umożliwiająca przeglądanie świeżego asortymentu i składanie zamówień w kilku kliknięciach.

[![Build Status](https://img.shields.io/badge/build-passing-brightgreen)](https://github.com/Johngtka/Piekarzynka)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

## 🔥 Główne Funkcje

* **Quick Order:** System zamawiania z wyborem godziny odbioru.
* **User Profile:** Historia zamówień i zbieranie punktów lojalnościowych.
* **Dark Mode:** Pełne wsparcie dla trybu ciemnego.
